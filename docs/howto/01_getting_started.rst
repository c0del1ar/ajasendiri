Getting Started
===============

Build
-----

.. code-block:: bash

   make

Run a file
----------

.. code-block:: bash

   ./ajasendiri my_program.aja

Type-check only
---------------

.. code-block:: bash

   ./ajasendiri check my_program.aja

Run tests
---------

.. code-block:: bash

   make test

Your first program
------------------

.. code-block:: text

   print("hello, world")

Save as ``hello.aja`` and run:

.. code-block:: bash

   ./ajasendiri hello.aja
